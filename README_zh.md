# Dust

Dust 是一款运行在 macOS 平台上的《魔兽争霸 III》（Warcraft III）录像解析工具。它能够解析 2.0.2 及以上版本的录像数据，并提供中英双语支持。

## 功能特性

* **版本兼容**：支持魔兽争霸 III 2.0.2+ 版本的录像文件。
* **平台支持**：原生支持 macOS。
* **多语言**：界面支持中文和英文切换。
* **信息解析**：提取玩家基本信息、APM、英雄数据及操作记录。

## 安装说明

由于本项目目前为个人分发版本，未在 Apple 开发者中心进行签名认证，安装后请按以下步骤操作以正常运行：

1. 下载 `.dmg` 文件并将 **Dust** 拖入 **Applications**（应用程序）文件夹。
2. 首次启动时，如果系统提示“无法打开”或“开发者身份不明”，请点击“取消”。
3. 打开 **系统设置 (System Settings)** -> **隐私与安全性 (Privacy & Security)**。
4. 找到“安全性”栏目，在提示“已阻止使用 Dust”下方点击 **仍要打开 (Open Anyway)**。
5. 在弹出的对话框中输入密码或使用 Touch ID，并再次点击 **打开**。
