# Dust

Dust is a Warcraft III replay parser built for macOS. It supports replay files from version 2.0.2 and above, with a focus on simplicity and accuracy.

## Features

* **Compatibility**: Supports Warcraft III replay versions 2.0.2+.
* **Platform**: Native macOS application.
* **Localization**: Available in both English and Chinese.
* **Data Parsing**: Extracts player information, APM, hero stats, and action logs.

## Installation

As this is an individually distributed version without an Apple Developer certificate, please follow these steps to install and run the app:

1. Download the `.dmg` file and drag **Dust** into your **Applications** folder.
2. When launching for the first time, if you see a warning stating "App cannot be opened because it is from an unidentified developer," click **Cancel**.
3. Open **System Settings** -> **Privacy & Security**.
4. Scroll down to the **Security** section and click **Open Anyway** next to the message regarding Dust.<img width="1430" height="996" alt="image" src="https://github.com/user-attachments/assets/1147db98-799b-41e0-92a3-574d302c620c" />

5. Authenticate with your password or Touch ID, then click **Open** again to confirm.
